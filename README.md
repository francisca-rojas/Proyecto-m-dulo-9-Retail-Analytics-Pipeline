#  Retail Analytics Pipeline con Apache Spark

##  Descripción del proyecto

Este proyecto desarrolla un pipeline completo de analítica de datos en un contexto de **Big Data**, utilizando Apache Spark para procesar grandes volúmenes de datos de retail y aplicar modelos de Machine Learning escalables.

El objetivo principal es transformar datos transaccionales en **información accionable**, permitiendo segmentar clientes y predecir aquellos de alto valor para optimizar estrategias de marketing.

---

## Objetivos

- Procesar grandes volúmenes de datos en un entorno distribuido  
- Generar métricas de negocio a partir de datos transaccionales  
- Implementar modelos de Machine Learning escalables  
- Obtener insights relevantes para la toma de decisiones  

---

##  Tecnologías utilizadas

-  Python  
-  Apache Spark (RDDs, DataFrames, Spark SQL)  
-  Spark MLlib  
-  Pandas / Matplotlib / Seaborn  
-  Google Colab  

---

##  Dataset

Se utilizó un dataset de transacciones de retail (e-commerce), que incluye información como:

- ID de cliente  
- Fecha de compra  
- Cantidad de productos  
- Precio unitario  
- País  
- Devoluciones  

Este dataset permite analizar el comportamiento de compra y construir variables agregadas por cliente.
